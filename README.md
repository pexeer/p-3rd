# p::3rd

build some open source c++ package automatically.

depends on package
- g++
- make
- cmake

 
## support to install package
- gtest
- gflags
- protobuf
- protobuf3
- zlib
- isa-l


## how to use
```
# only install protobuf3
./build.3rd protobuf3

# this command will install gtest gflags protobuf3
./build.3rd
```
